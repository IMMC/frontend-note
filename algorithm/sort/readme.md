# 排序算法

## 时间复杂度为 O(n^2) 的算法

1. 冒泡排序
2. 选择排序
3. 插入排序
4. 希尔排序

## 时间复杂度为 O(nlogn)

1. 快速排序
2. 归并排序
3. 堆排序

## 时间复杂度线性增加

1. 计数排序
2. 桶排序
3. 基数排序

## 排序算法稳定性

通过相同值的元素排序后 保持原有位置顺序不表 则被认为是稳定排序，如果位置发生变化则认为是不稳定排序。单纯数字排序，稳定性 完全没有意义。

### 稳定的排序算法

1. 冒泡排序
2. 插入排序
3. 归并排序
4. 计数排序
5. 桶排序
6. 基数排序

### 不稳定的排序算法

1. 选择排序
2. 快速排序
3. 堆排序
4. 希尔排序
